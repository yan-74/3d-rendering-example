# Three.js Rendering Pipeline Workshop

This is a simple Three.js demo that demonstrates the 3D rendering pipeline!

Pipeline:

Setup → Geometry → Vertex Shading → Rasterization → Fragment Shading

## How to Run

1. Download or clone the repo
2. Open the `starter` folder
3. Open `index.html` with Live Server
4. Follow the workshop instructions
5. Paste code step by step

## Final Result

A rotating cube with lighting that demonstrates the rendering pipeline.
